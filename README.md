

### **Agenda 1: C++ Basics for 15-Year-Olds**

**Duration**: 6 Weeks

#### **Week 1: Introduction to Programming**
- **Objective**: Introduce students to programming and C++.
- **Topics**:
  - What is programming?
  - Introduction to C++
  - Setting up a development environment
  - Writing your first C++ program: "Hello, World!"
- **Activities**:
  - Print text exercises.
  - Explore syntax rules.

#### **Week 2: Variables and Data Types**
- **Objective**: Understand variables and data types in C++.
- **Topics**:
  - Variables: declaration and initialization.
  - Data types: int, float, char, string.
  - Type conversion.
- **Activities**:
  - Write programs to use and manipulate variables.
  - Create a temperature converter (Celsius to Fahrenheit).

#### **Week 3: Input and Output Operations**
- **Objective**: Learn how to handle input and output in C++.
- **Topics**:
  - Using `cin` and `cout` for input and output.
  - Formatting output.
  - Basic mathematical operations.
- **Activities**:
  - Create a simple calculator.
  - Program to take user input and display a personalized message.

#### **Week 4: Operators and Expressions**
- **Objective**: Get familiar with different operators in C++.
- **Topics**:
  - Arithmetic, relational, and logical operators.
  - Compound expressions.
  - Operator precedence.
- **Activities**:
  - Write a program to compare two numbers.
  - Explore logical operations through conditional checks.

#### **Week 5: Control Structures (Part 1)**
- **Objective**: Master decision-making using control structures.
- **Topics**:
  - If-else statements.
  - Nested if-else.
  - Switch-case.
- **Activities**:
  - Write a program to find the largest of three numbers.
  - Implement a basic menu-driven program.

#### **Week 6: Control Structures (Part 2)**
- **Objective**: Learn loops and repetitive tasks in C++.
- **Topics**:
  - For loop, while loop, and do-while loop.
  - Break and continue statements.
- **Activities**:
  - Write a program to calculate the sum of the first 10 natural numbers.
  - Create a multiplication table generator.

### **Agenda 2: Data Structures in C++ for 15-Year-Olds**

**Duration**: 6 Weeks

#### **Week 1: Introduction to Arrays**
- **Objective**: Learn about arrays and basic operations.
- **Topics**:
  - What is an array?
  - Declaring and initializing arrays.
  - Accessing array elements.
- **Activities**:
  - Write a program to store and display student grades.
  - Implement a program to find the maximum value in an array.

#### **Week 2: Strings and String Operations**
- **Objective**: Explore string handling in C++.
- **Topics**:
  - Declaring and initializing strings.
  - Common string operations (concatenation, comparison, length).
- **Activities**:
  - Create a program to reverse a string.
  - Implement a simple text-based hangman game.

#### **Week 3: Sorting and Searching Algorithms**
- **Objective**: Introduce basic sorting and searching techniques.
- **Topics**:
  - Bubble sort, selection sort.
  - Linear search and binary search.
- **Activities**:
  - Write a program to sort an array of numbers.
  - Implement a program to search for a specific value in an array.

#### **Week 4: Introduction to Linked Lists**
- **Objective**: Understand linked lists and their operations.
- **Topics**:
  - Concept of linked lists.
  - Creating and traversing a linked list.
  - Insertion and deletion operations.
- **Activities**:
  - Write a program to create and display a linked list.
  - Implement insertion at the beginning and end of a linked list.

#### **Week 5: Stacks and Queues**
- **Objective**: Learn about stacks and queues in C++.
- **Topics**:
  - Concept of stacks (LIFO) and queues (FIFO).
  - Stack operations: push, pop, peek.
  - Queue operations: enqueue, dequeue, front.
- **Activities**:
  - Write a program to implement a stack using arrays.
  - Implement a queue using linked lists.

#### **Week 6: Recursion and Advanced Data Structures**
- **Objective**: Introduce recursion and advanced data structures.
- **Topics**:
  - What is recursion?
  - Recursive functions in C++.
  - Introduction to trees (if time permits).
- **Activities**:
  - Write a recursive program to calculate the factorial of a number.
  - Implement a basic binary tree with insertion and traversal (optional).

### **Final Project for C++ Basics**

#### **Project Title**: **Simple Banking System**

#### **Objective**: Apply fundamental C++ concepts to create a basic banking system.

#### **Project Description**:
Create a console-based application that simulates basic banking operations. The system should allow users to create an account, deposit money, withdraw money, and check their balance.

#### **Features**:
1. **Account Creation**:
   - Users can create an account by providing their name and an initial deposit amount.
   - The system should generate a unique account number.

2. **Deposit Money**:
   - Users can deposit money into their account.
   - The system should update the account balance accordingly.

3. **Withdraw Money**:
   - Users can withdraw money from their account.
   - Ensure that the withdrawal does not exceed the account balance.

4. **Check Balance**:
   - Users can check their account balance at any time.

5. **Exit**:
   - The program should provide an option to exit.

#### **Deliverables**:
- **Source Code**: The complete C++ code for the banking system.
- **Documentation**: A brief document explaining how the program works and its features.
- **Screenshots**: Demonstrating each feature (e.g., account creation, deposit, withdrawal).

### **Final Project for Data Structures in C++**

#### **Project Title**: **Library Management System**

#### **Objective**: Utilize various data structures to manage a small library system.

#### **Project Description**:
Develop a console-based library management system that uses arrays, linked lists, and/or other data structures to manage books and users.

#### **Features**:
1. **Book Catalog**:
   - Implement a list (array or linked list) of books where each book has a title, author, and an ID.
   - Allow users to add, remove, and search for books.

2. **User Accounts**:
   - Implement a list of users, each with a unique ID and name.
   - Allow users to register and delete accounts.

3. **Book Borrowing**:
   - Implement a system where users can borrow and return books.
   - Track which books are borrowed by which user.
   - Ensure that a book cannot be borrowed if it is already checked out.

4. **Display Catalog and User List**:
   - Allow the user to view the current list of books and registered users.

5. **Data Persistence (Optional)**:
   - Save the library data to a file and load it when the program starts.

#### **Deliverables**:
- **Source Code**: The complete C++ code for the library management system.
- **Documentation**: An explanation of the data structures used and how the program works.
- **Screenshots**: Demonstrating key functionalities like adding books, registering users, and borrowing books.

### **Final Project for HTML Basics**

#### **Project Title**: **Personal Portfolio Website**

#### **Objective**: Design a multi-page personal portfolio using HTML to showcase the skills learned.

#### **Project Description**:
Create a static personal portfolio website consisting of multiple pages that demonstrate the ability to create and link different types of content using HTML.

#### **Features**:
1. **Home Page**:
   - A welcoming homepage that includes an introduction and a brief biography.
   - Use headings, paragraphs, and images to make the page visually appealing.

2. **Projects Page**:
   - A page that lists your projects, with each project having a title, description, and link to further details or a demo.
   - Use ordered/unordered lists and links.

3. **Contact Page**:
   - A contact form where visitors can enter their name, email, and a message.
   - Include various input types (text, email, textarea).

4. **About Me Page**:
   - A dedicated page with detailed information about your background, skills, and hobbies.
   - Use images, lists, and links to enhance the content.

5. **Navigation**:
   - Ensure that all pages are interlinked with a navigation menu at the top or side of each page.
   - Use anchor tags and consistent styling for easy navigation.

6. **Optional (Styling and Interactivity)**:
   - Add basic CSS for styling.
   - Use internal or external stylesheets for consistent design across pages.

#### **Deliverables**:
- **HTML Files**: All the HTML files for the website.
- **Documentation**: A brief document describing the structure of the website and its contents.
- **Screenshots**: Showing each page of the website.
- **Live Demo (Optional)**: Host the website on a free web hosting service like GitHub Pages and provide a link.
